# AlgoBridge Browser Extension

The bridge between your browser and VS Code.

## Installation

1. Load this folder as an "Unpacked Extension" in Chrome.
2. Pin the icon to your toolbar.

## Usage

1. Open a problem on Codeforces.
2. Click the icon to send the problem data to VS Code.

## Note

This extension requires the **AlgoBridge VS Code Extension** to be running locally to receive and display data.
